## ğŸ”§ TROUBLESHOOTING

### Problem: "Can't connect" error
**Solution:** 
- Make sure `START_APP.bat` is running
- Check the black window is open
- Try `localhost:8000` again

### Problem: Products not showing
**Solution:**
- Click "Manage Products"
- Add products first
- They'll appear immediately

### Problem: Bill won't print
**Solution:**
- Check printer is on
- Check printer is connected
- Try "Save as PDF" instead

### Problem: Data disappeared
**Solution:**
- Don't clear browser data
- Don't use incognito mode
- Data saves automatically in normal mode

### Problem: Can't type numbers
**Solution:**
- Click outside search box first
- Make sure you're not in any input field
- Then type numbers
